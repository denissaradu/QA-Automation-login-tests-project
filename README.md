# This project demonstrates basic QA automation concepts such as Page Object Model, Pytest fixtures, and Selenium WebDriver.
# QA Automation Login Tests

This project contains automated UI tests for the login functionality of the SauceDemo website using Selenium and Pytest.

## Website tested
https://www.saucedemo.com/

## Technologies used
Python  
Selenium  
Pytest  

## Project structure

pages/ - Page Object Model classes  
tests/ - automated test cases  
conftest.py - browser setup fixture  

## Tests included

1. Valid login test
2. Invalid login test
3. Logout test

## How to run the tests

Install dependencies:

pip install -r requirements.txt

Run tests:

pytest
